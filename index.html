The Geospatial data and services portal facilitates the coordinated development, maintenance, and sharing of geospatial data and web map services by Norwegian Polar Instiute.

To meet the ever increasing number of tile-based queries against forum Administration WMS services, we now offer a selection of our basic mapping services that cache services. The services can be used in clients with support for WMS-C as OpenLayers, Google Maps or Microsoft Virtual Earth The purpose of the cache is to achieve fast response times of map requests. A cache service is not as flexible as for example, a WMS service. For example, when a WMS service could select one or more layers while in a cache must select all or nothing. Authority uses free software component GeoWebCache to caching services.
