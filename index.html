Norwegian Polar Institute Maps and Services

  • Base Maps

Norwegian Polar Institute provides free access to a wide range of basemap and thematic map services for the Nowegian polar regions which can be consumed to create interactive web mapping application. The basemaps are cached map services while the thematic maps are dynamic map services.

Cached Map Services

Cached map service is a regular map service that has been predrawn by the server at defined scales (zoom levels) in order to serve maps very quickly.

NPI offers a selection of basemap services that are cached services. The services can be used in clients with support for ArcGIS Server tiles such as OpenLayers, Google Maps or Esri's JavaScript API in the same way as Google Maps or Esri Maps is built up.
Read the developer's guide on how to create web applications with those basemaps. The following basemaps are provided as cached map services:

Zoom levels and scales

To easily combine cache from different services in the same map window, there is an advantage to have a common definition of the geographic area that defines the service. Because the cached services have fixed zoom levels, the service can not deliver tiles that lies between two defined zoom levels.

Image formats and coordinate systems

Services are available in two image formats:

  • Image / jpeg (does not support transparency)
  • Image / png (supports transparency)

Tile size for jpeg is about 1/3 of the png so it must be considered what is important, speed or support for transparency. The image tiles have a resolution of 96 dpi with a size of (256x256 pixels).

Terms of Use

It's free to use Norwegian Polar Institute's map services, but remember to read the terms of use of NPI's map services (Norwegian).

Thematic Map Services

Polar institute delivers a wealth of thematic maps as dynamic map services. Dynamic map service is a service where a server request to send map section in the form of raster files back to the user, either directly in the browser, a web application or a desktop application. This type of service requires some expertise to take in use, and it's the application that makes calls to map server.

Services based on WMS specification are flexible and provides the user with many options, such as:

  • Choosing which map projection will be produced in
  • Turn on or off layers in the service.
  • Choose scale freely
  • Which image format and transparency to be produced
  • Retrieving Legend of the various map layers
  • Obtaining basic information about objects on the map from attributes (feature info) etc

Checkout also the list of map services provided by NPI on the ArcGIS Server REST API.

  • Svalbardkartet

    Svalbardkartet is an interactive thematic atlas of Svalbard supplemented with a rich datasets of environmental data layers with advanced map viewer. In addition to the detailed topography and large amount of thematic maps, the portal provides advanced GIS capabilities such as high-resolution map printing, identifiying map objects, advanced drawing and measurements

  • Barentsportal

    Barentsportal is an interactive thematic atlas of the joint Norwegian-Russian environmental status report for the barents sea.

  • TopoSvalbard

    TopoSvalbard is an interactive topographical map portal of Svalbard. In addition to the presentation of detailed topography and placenames, the portal includes basic user requirements such as coordinate readings and distance measurements. It also includes 3D map of Svalbard as well as areal and landscape photos

  • TopoJanMayen

    TopoJanMayen is an interactive topographical map of Jan Mayen.
